
### 1. 标题 (Headings)

在文字前面加 `#` 号，注意 `#` 和文字之间要有一个**空格**。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

- `# 一级标题` (最大的标题)
    
- `## 二级标题`
    
- `### 三级标题` (以此类推，最多支持六级)
    

### 2. 字体格式 (Text Styles)

- **加粗**：在文字两边各加两个星号，如 `**这是加粗**`。
    
- _斜体_：在文字两边各加一个星号，如 `*这是斜体*`。
    
- ~~删除线~~：在文字两边各加两个波浪号，如 `~~这是删除线~~`。
    
- ==高亮==：在文字两边各加两个等号，如  `==这是高亮==`

### 3. 列表 (Lists)

- **无序列表**：输入 `-` 或 `*` 加空格。
    
    - 列表项 A
        
    - 列表项 B
        
- **有序列表**：输入 `1.` 加空格。
    
    1. 第一步
        
    2. 第二步
        
- **待办事项 (Task List)**：输入 `- [ ]` 加空格。
    
    - [ ] 未完成任务
        
    - [x] 已完成任务  
    -  (打个 x 即可)  
        
### 4. 引用与代码 (Quotes & Code)

- **引用块**：输入 `>` 加空格。
    
    > 哪怕是再小的帆，也能远航。
    
- **行内代码**：用反引号包裹，如 `` `Ctrl + C` ``。
    
- **代码块**：用三个反引号包裹。
    

Python

```
print("Hello Obsidian")
```

### 5. 图片与链接 (Links)

- **内部链接**（Obsidian 的灵魂）：输入 `[[笔记名称]]`。这能让你双击直接跳转到另一篇笔记。
    
- **外部链接**：`[显示文字](网址)`。
    
- **插入图片**：`![图片描述](图片链接或本地路径)`。
